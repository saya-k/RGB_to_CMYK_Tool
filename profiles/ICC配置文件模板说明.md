# ICC 配置文件模板说明

这个文件夹用于放置印刷厂提供的 CMYK ICC 配置文件。

## 重要说明

ICC 文件不是普通文本配置文件，而是专业软件生成的二进制色彩配置文件。

请不要把本说明文件改名为 `.icc` 使用，也不要使用来历不明的 ICC 文件直接作为正式印刷标准。

## 需要印刷厂提供什么

请向印刷厂索要以下文件之一：

- `.icc`
- `.icm`

并确认它是：

- CMYK 色彩空间
- 适用于当前纸张
- 适用于当前印刷工艺
- 适用于当前油墨总量限制，也叫 TAC 或 Total Area Coverage

## 推荐向印刷厂确认的信息

可以把下面这段发给印刷厂：

```text
请提供本批包装/产品图片印刷使用的 CMYK ICC 配置文件。

用途：RGB 图片批量转换 CMYK。
文件格式：.icc 或 .icm。
色彩空间：CMYK。
纸张类型：
印刷工艺：
油墨总量限制 TAC：
适用标准：
备注：
```

## 常见字段解释

- Profile Name：配置文件名称，例如 Coated Fogra39L_300%。
- Color Space：色彩空间，正式印刷通常需要 CMYK。
- Paper Type：纸张类型，例如铜版纸、哑粉纸、胶版纸、牛皮纸。
- TAC：总墨量限制，例如 300%。数值越高，暗部油墨越重，也越容易干燥慢、糊版或脏。
- Standard：印刷标准，例如 FOGRA39、FOGRA51、GRACoL、SWOP 等。
- Region：地区标准，例如 Europe、US、Japan。

## 放置方法

把印刷厂提供的 `.icc` 或 `.icm` 文件直接放进本文件夹：

```text
RGB_to_CMYK_Tool/profiles/
```

工具运行时会优先使用这里的 CMYK ICC 文件。

如果这里没有可用 ICC 文件，工具会使用通用 CMYK 转换，并在日志里提示：

```text
未使用印刷厂ICC，仅为通用转换
```

## 示例

网站或软件里常见的 `Coated Fogra39L_300%` 大致表示：

- Coated：涂布纸、铜版纸类
- FOGRA39：欧洲常用印刷标准
- 300%：TAC 总墨量上限为 300%

这类文件是否适合你的包装印刷，要以印刷厂要求为准。
